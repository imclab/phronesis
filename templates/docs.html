data api

The general scheme is base_url/<version>/<username>/<l1_data_category>/<l2_data_category>/<resource> ? <parameters>

Parameters are: start_date, end_date, location (lat, lon), radius, users, descriptive_stats (mean, med, mode, sd).

example: localhost:8000/v1/pdarche/body/nutrition/meals?start_date=101010&end_date=101020&location=home&r=500
